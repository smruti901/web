Introduction

1. Inheritance

The process of acquiring the properties from one class to another is called inheritance. We can achieve inheritance by using "extends" keyword.

Types of Inheritance :

1. Single Level

2. Multilevel

3. Multiple

4. Hierarchical

5. Hybrid

2. Polymorphism

It is defined as an ability of an object which possess different behaviour in diffrent situation.

It is of two types :

1. Compiletime Polymorphism

2. Runtime Polymorphism

3. Encapsulation

The process of binding data members and member function into a single unit.

To provide more Encapsulation we need to take the help of private variable and access it outside the class with the help of "getters" and "setters".

4. Abstraction

The process of hiding the implementation details and providing the functionality to the users.

We can achieve Abstraction by using abstarct class and interface. Based on the method implementation it is classified into two types

1. Complete or Concrete Method

2. Incomplete or Abstract Method

5. Interface

It should be defined with "interface" keyword. It allows only incomplete method and it doesn't have constructor inside it.

To use interface inside the class we need to take help of "implements" keyword. To achieve inheritance between two interfaces we have to use "extends" keyword.

Syntax : public interface Interfacename
