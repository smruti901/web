Introduction

In collections we can store both homogeneous and heterogeneous elements. Collections are growable in nature. All the collections are availbe under "java.util" package.

Linkedlist

It follows double linked list. Duplicate objects and null insertion is allowed.

For each loop

It is an enhanced for loop. It is mainly used for collections and array.

Syntax : for(datatype variable : collection) { }

Iterator

It is predefined interface in "java.util" package. It is used to perform cursor operation

Methods of Iterator :

1. iterator() - It is used to arrange the elements based on insertion order

2. hasNext() - It is used to check the availability of next element in the iterator

2. next() - It is used to fetch the current element value and after fetching it will move the pointer to next element.
